- Preload scripts expose a typed bridge via `contextBridge`; update `src/preload/index.d.ts` when adding IPC methods.
- Prefer `zustand` stores for renderer state; keep main-process state in modules under `src/main/`.

- TypeScript strict mode via `@electron-toolkit/tsconfig`. Run `npm run typecheck` before closing a task.
- Vitest with jsdom. Tests live next to source (co-located). Use `@testing-library/react` for renderer tests.
